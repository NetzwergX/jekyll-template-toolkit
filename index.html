Jekyll Boilerstrap

A blogging toolkit for static sites, 100% compatible to GitHub Pages.

Built on top of Twitter-Bootstrap & Jekyll, utilizies Font-Awesome.

Download
or Fork 
$ git clone https://github.com/NetzwergX/jekyll-boilerstrap.git Username.github.io
$ cd Username.github.io
$ git remote set-url origin git@github.com:Username/Username.github.io
$ git push origin master

Hosting on GitHub

Jekyll Boilerstrap is 100% compatible with GitHub Pages. You can push a site crafted with Jekyll-Boilerstrap to a GitHub Pages endpoint and it will built automatically, giving you a free and easy hosting solution for your site!
