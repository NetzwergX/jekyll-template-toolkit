Pages

List of pages included in Jekyll Boilerstrap

  • Post Index
  • Post Archive (time-based)
  • Category List
  • Tag List
  • User Sitemap
  • XML-based meta pages
  • Atom-Feed
  • RSS-Feed
  • Sitemap

Some pages require certain Jekyll settings to be set properly, others offer custom settings that can be configured through custom variables. Please check the list of settings for each page. If you use the _config.yml deployed within J-BS, all pages should work out of the box.

Indexing pages

Post Index

List of all posts sorted by their publishing date in descending order. Optional pagination available.

paginate: 5 		# needs to be positive non-zero value
pagination: pager 	# possible values are pager | pagination
